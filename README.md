# 🏬 Super Store Sales Dashboard

An interactive **Super Store Sales Dashboard** built to analyze sales performance, profitability, and customer trends using data visualization techniques.

---

## 📊 Overview

This dashboard provides a comprehensive analysis of:
- Total Sales, Profit, and Quantity
- Regional and state-wise performance
- Category & Sub-category insights
- Customer segment analysis
- Shipping and order trends

---

## 🚀 Features

- 📈 Interactive and dynamic visualizations
- 🌍 Region & state-wise sales analysis
- 🛍️ Category & sub-category performance tracking
- 👥 Customer segment insights (Consumer, Corporate, Home Office)
- 🚚 Shipping mode analysis
- 📅 Time-based trends (monthly/yearly)

---

## 🛠️ Tools & Technologies Used

- Power BI / Tableau / Excel / SQL
- Data Cleaning & Transformation
- Data Visualization

---

## 📂 Dataset

- Dataset used is the popular **Super Store dataset (open-source platform's)**
- Contains:
  - Order details
  - Customer segments
  - Product categories & sub-categories
  - Sales, profit, and discount data

## 📌 Key Insights

- Technology category generates the highest sales
- Office Supplies contribute steady profits
- Some sub-categories show losses due to high discounts
- Western region leads in overall sales
- Consumer segment contributes the most revenue
